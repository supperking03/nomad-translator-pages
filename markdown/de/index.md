# Menüs, Schilder, Nachrichten und Gespräche ohne Internet übersetzen.

Nomad Translator hilft Reisenden auf dem iPhone mit Text-, Sprach- und Kameraübersetzung, nachdem ein Sprachpaket einmal geladen wurde.

## Product Summary

Nomad Translator is an iPhone travel translator for offline-first use cases: text translation, voice input, camera/photo translation, downloadable language packs, and on-device processing.
