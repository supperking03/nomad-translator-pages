# Translate menus, signs, messages, and conversations without the internet.

Nomad Translator helps travelers translate on iPhone with text, voice, and camera input after downloading a language pack once.

## Product Summary

Nomad Translator is an iPhone travel translator for offline-first use cases: text translation, voice input, camera/photo translation, downloadable language packs, and on-device processing.
